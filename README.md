### Structure du projet
```
portfolio/
├── index.html          → Page d'accueil
├── pages/
│   ├── about.html      → À propos (parcours, compétences)
│   ├── contact.html    → Formulaire de contact
│   └── gallery.html    → Galerie de projets
├── css/
│   └── style.css       → Toute la mise en forme
└── images/             → Mettre vos images ici
```

### Comment ajouter votre photo ?
1. Mettez votre photo dans le dossier `images/` (ex: `photo.jpg`)
2. Dans `index.html`, remplacez :
   ```html
   <div class="hero-photo" ...>👩🏾‍💻</div>
   ```
   par :
   ```html
   <img class="hero-photo" src="images/photo.jpg" alt="Photo de Khadidiatou DIANE">
   ```

### Technologies utilisées
- HTML5 (balises sémantiques)
- CSS3 (Flexbox, variables CSS, animations, media queries)
- Responsive Design Mobile First
- Aucun JavaScript, aucun Bootstrap
